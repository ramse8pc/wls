Oracle WebLogic Server
======================

The following supplemental or dependent software is required for the successful setup of
the Oracle WebLogic Server (WLS) software stack:

  - Oracle JDK 1.7u51 (WLS 10.3.6 and higher)
    - jdk-7u51-linux-x64.tar.gz

  - Oracle JRockit 1.6u45-R28.2.7-4.1.0 (WLS 10.3.5)
    - jrockit-jdk1.6.0_45-R28.2.7-4.1.0-linux-x64.bin
